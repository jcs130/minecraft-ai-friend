# 路线图

## 0.1 MVP

- 中文产品化首页。
- 一键启动陪玩流程。
- Minecraft Server 管理。
- Mindcraft 配置整合。
- Agent Profile 编辑。
- 自动陪玩基础循环。

## 0.2 新手任务系统

- 第一晚生存任务进度条。
- 任务完成状态检测。
- 新手提示和下一步建议。
- 基地位置和重要坐标记忆。

## 0.3 AI 队友产品化

- 角色模板：导师、护卫、建筑伙伴、探索伙伴、管家。
- 每个角色独立记忆和行为边界。
- 多 Agent 管理。

## 0.4 语音和客户端体验

- TTS/STT 整合。
- 客户端 Sidecar。
- 语音唤醒和语音指令。

## 0.5 客户端/Mod 能力

- Fabric/Forge Mod 获取玩家状态、背包、附近实体和坐标。
- 更可靠的游戏内事件判断。
- 客户端侧音频播放和 UI 提示。

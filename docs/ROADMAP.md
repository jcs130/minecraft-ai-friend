# 路线图

## 0.1 MVP：已完成

- 中文产品化首页。
- 一键启动陪玩流程。
- Minecraft Server 管理。
- Mindcraft 配置整合。
- Agent Profile 编辑。
- 自动陪玩基础循环。

## 0.2 新手任务系统：进行中

- 第一晚生存任务模板。
- 新手提示和下一步建议。
- 基地位置、公共箱子和重要坐标记忆。
- 任务完成/受阻事件流。

## 0.3 AI 队友产品化：进行中

- 默认五居民角色：Alex、Luna、Milo、Nova、Ivy。
- 每个角色独立记忆和行为边界。
- AI 村长 Airi 的全局任务派发。
- 中文公开行动思考和观众可见字幕清理。

## 0.4 数据和记忆系统：进行中

- SQLite 事件库。
- Agent 状态上报和长期记忆。
- SQLite 本地向量缓存。
- Ollama/OpenAI-compatible embedding。
- 可选 Qdrant。

## 0.5 直播和观察

- 统一直播可视化页面。
- `live` / `ServerTV` 观察账号自动聚焦活跃 AI。
- AI 思考字幕、任务事件、资源和设施展示。
- OBS 推流工作流。

## 0.6 客户端/服务端插件能力

- Fabric/Forge/服务端插件获取玩家状态、背包、附近实体和坐标。
- 更可靠的游戏内事件判断。
- 箱子库存、方块变更、死亡、区域状态同步到事件库。
- 客户端侧音频播放和 UI 提示。

## 0.7 多 Agent 社会系统

- 任务主表和任务验收。
- 居民关系、偏好和长期计划。
- 村长巡查、直播解说和观众问答。
- Postgres/pgvector 或 Qdrant 的多人/直播部署方案。

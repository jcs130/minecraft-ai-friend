# 产品设计

## 产品定位

“我的世界AI陪玩”是给新玩家的本地/局域网 Minecraft AI 队友。它把开服、启动 Mindcraft、AI 队友陪玩、新手任务、服务器管理和高级配置整合到一个中文网页里。

## 目标用户

- 第一次接触 Minecraft Java 服务器的新玩家。
- 想在局域网里和朋友、孩子一起玩的房主。
- 希望让 AI 队友陪玩、教学、打怪、建造和探索的玩家。
- 想使用 Mindcraft 但不想手改 settings.js/profile JSON 的用户。

## 核心体验

1. 打开网页。
2. 点击“一键启动陪玩”。
3. 服务器和 Mindcraft 依次启动。
4. AI 队友进服。
5. 用户选择“第一晚生存”“新手导师”“生存护卫”“建筑伙伴”等陪玩模板。
6. AI 在游戏聊天中解释、协作和行动。

## MVP 功能

- 一键启动 Minecraft Server、Mindcraft 和自动陪玩。
- 准备状态清单：服务器、Mindcraft、AI 队友、自动陪玩。
- 陪玩模板：第一晚生存、新手导师、生存护卫、建筑伙伴。
- 快速指令：找食物、打怪、改善基地、探索、回基地、自由活动。
- 高级控制：服务器配置、Mindcraft 配置、Agent Profile、日志和控制台命令。

## 后续方向

- 客户端 Sidecar。
- Fabric/Forge Mod 获取更丰富客户端状态。
- 语音对话和客户端侧语音播放。
- 多 AI 队友队伍。
- 新手任务包和世界模板。
- 家长/儿童安全模式。
